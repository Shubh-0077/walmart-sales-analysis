# Walmart Sales Data Analysis

## 📊 Project Overview

This project analyzes Walmart sales data to uncover business insights using Python (Pandas) and SQL. The workflow includes data cleaning, exploratory data analysis (EDA), and answering business questions using SQL queries.

## 🎯 Objectives

* Clean and prepare raw sales data
* Perform exploratory data analysis (EDA)
* Identify sales trends and patterns
* Answer business questions using SQL
* Generate actionable insights

## 🛠 Tools Used

* Python (Pandas, NumPy)
* Jupyter Notebook
* SQL (PostgreSQL / MySQL)
* Data Visualization (Matplotlib / Seaborn)

## 🔄 Workflow

1. Data cleaning using Python Pandas
2. Exploratory Data Analysis (EDA)
3. Feature understanding and transformation
4. Load cleaned data into SQL database
5. Answer business questions using SQL queries
6. Extract insights from results
